# KWEON SOONYEOB — QA Engineer Portfolio

원인과 위험을 추적하는 QA 엔지니어 권순엽의 포트폴리오입니다.

## 실행

```bash
npm ci
npm run dev
```

개발 서버의 `/portfolio` 경로에서 확인할 수 있습니다.

## 배포

`main` 브랜치에 변경사항을 push하면 GitHub Actions가 정적 사이트를 빌드하고
GitHub Pages에 자동 배포합니다.

- Production: https://syeob.github.io/portfolio/
- Node.js: 22
